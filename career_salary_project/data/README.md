# data/ 폴더

여기에 분석할 데이터 파일을 넣으세요.

> `sample.csv`는 **바로 동작 확인용 예시 데이터**(학생 합격 예측, 320행)입니다.
> 템플릿이 잘 도는지 확인한 뒤, **본인 데이터로 교체**하세요.
>
> 이미지 프로젝트면 `images/<클래스명>/` 구조로 둡니다(샘플 cat/dog 포함). → `../이미지_데이터_쓸_때/README.md` 참고.

- `src/data_loader.py`의 `load_data()`가 이 폴더에서 파일을 읽습니다.
- 파일이 아주 크면(수백 MB↑) 제출 시 데이터는 빼고 **출처 URL을 보고서에 적으세요.**
- 출처/라이선스를 보고서에 반드시 표기.
