# 6주차 1교시 — Ollama 소개 및 로컬 LLM 기초

## 실습 코드 목록

| 실습 | 파일명 | 내용 | 실행 방법 |
|------|--------|------|----------|
| 실습 3 | `ollama_basic.py` | `ollama.generate()` — 단순 텍스트 생성 | `python ollama_basic.py` |
| 실습 4 | `ollama_chat.py` | `ollama.chat()` — 대화형 API (메시지 리스트) | `python ollama_chat.py` |
| 실습 5 | `ollama_multi_turn.py` | 시스템 프롬프트 + 다중턴 대화 | `python ollama_multi_turn.py` |
| 실습 6 | `ollama_temperature.py` | Temperature 비교 실험 (0.0 / 0.7 / 1.5) | `python ollama_temperature.py` |

## 사전 준비

1. **Ollama 서버 실행 확인** — 작업 표시줄 트레이에 Ollama 아이콘이 있는지 확인
2. **모델 설치 확인** — 터미널에서 `ollama list` 실행 → `gemma3:4b`가 목록에 있어야 함
3. **Python 패키지 설치** — `pip install ollama`

## 실습 순서

### 실습 3: ollama_basic.py
- `ollama.generate()`로 가장 기본적인 텍스트 생성
- `prompt` 파라미터에 문자열 하나를 전달하는 단일턴 방식

### 실습 4: ollama_chat.py
- `ollama.chat()`으로 대화형 API 사용
- `messages` 리스트에 `{"role": "user", "content": "..."}` 형태로 전달

### 실습 5: ollama_multi_turn.py
- 시스템 프롬프트(`role: system`)로 AI에 역할 부여
- 첫 번째 질문 → AI 답변 추가 → 두 번째 질문으로 다중턴 대화 구현
- `messages.append()`로 대화 기록을 누적하는 과정 체험

### 실습 6: ollama_temperature.py
- 동일한 프롬프트에 Temperature 0.0 / 0.7 / 1.5를 각각 3번씩 호출
- 총 9번 호출하므로 1~3분 소요될 수 있음
- Temperature 0.0은 매번 같은 답변, 1.5는 매우 다양한 답변이 나오는지 관찰

## 주의사항

- 모든 코드 실행 전 **Ollama 서버가 실행 중**이어야 합니다
- `ConnectionError` 발생 시 → Ollama 앱을 실행하거나 터미널에서 `ollama serve` 실행
