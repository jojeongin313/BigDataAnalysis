# 1. 데이터 선정

## 데이터명
Stack Overflow Developer Survey 2025

## 출처
- Stack Overflow Annual Developer Survey 2025
- Kaggle Dataset
- https://www.kaggle.com/datasets/aliaslam25/stack-overflow-developer-survey-2025

## 라이선스
- Open Database License (ODbL)
- 공개 설문 데이터

## 데이터 형식
- CSV

## 데이터 규모
- 전 세계 개발자 대상 설문조사 데이터
- 수만 명 규모의 응답자
- 100개 이상의 컬럼 포함

## 주요 컬럼

### 개발자 정보
- 직무(Developer Type)
- 국가(Country)
- 경력(Years of Experience)
- 근무 형태(Remote/Hybrid/Office)

### 기술 스택
- 프로그래밍 언어(Language)
- 데이터베이스(Database)
- 클라우드 플랫폼(Cloud Platform)
- 웹 프레임워크(Web Framework)

### AI 관련
- AI 도구 사용 여부
- AI Agent 사용 여부
- LLM 사용 현황
- AI 활용 방식

### 연봉 관련
- 연봉(Compensation)
- 통화(Currency)
- 국가별 연봉 정보

## 데이터 선정 이유

본 데이터는 전 세계 개발자를 대상으로 진행된 최신 설문조사 데이터로, 개발자의 직무, 기술스택, 연봉, AI 활용 현황 등 다양한 정보를 포함하고 있다.

특히 최근 개발자 채용 시장과 AI 기술 트렌드를 함께 분석할 수 있으며, 실제 개발자 데이터를 기반으로 한 추천 서비스 구현에 적합하다고 판단하여 선정하였다.

# 2. 풀고 싶은 문제 정의

## 문제 상황

개발자를 준비하는 학생 및 취업 준비생은 다음과 같은 어려움을 겪는다.

- 어떤 기술을 우선적으로 공부해야 하는지 알기 어렵다.
- 희망 직무별 요구 기술스택을 파악하기 어렵다.
- 실제 개발자들의 연봉 수준을 알기 어렵다.
- AI 시대에 어떤 기술이 유망한지 판단하기 어렵다.

## 해결하고 싶은 문제

실제 개발자 설문 데이터를 분석하여 직무별 기술스택과 연봉 정보를 제공하고, 사용자가 희망 직무를 선택하면 해당 직무에 적합한 기술스택과 자격증을 추천하는 서비스를 구현하고자 한다.

# 3. 프로젝트 목표

본 프로젝트는 Stack Overflow Developer Survey 2025 데이터를 활용하여 개발자 직무별 기술스택, 연봉, AI 활용 현황을 분석하고, 사용자가 희망 직무를 선택하면 해당 직무에 적합한 기술스택과 자격증을 추천하는 웹 서비스를 구현하는 것을 목표로 한다.

예를 들어 사용자가 Data Engineer를 선택하면 다음과 같은 정보를 제공한다.

- 추천 프로그래밍 언어
- 추천 데이터베이스
- 추천 클라우드 플랫폼
- 추천 자격증
- 평균 연봉
- AI 활용 현황
