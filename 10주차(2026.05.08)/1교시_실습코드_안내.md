# 7주차 1교시 — 웹 보안 기초 및 공격 유형 이해

## 실습 코드 목록

| 실습 | 파일명 | 내용 | 실행 방법 |
|------|--------|------|----------|
| 실습 1 | `attack_examples.py` | 정상 vs 공격 HTTP 요청 비교 + 규칙 기반 탐지기 만들기 | `python attack_examples.py` |
| 실습 2 | `attack_quiz.py` | 웹 공격 탐지 퀴즈 (대화형) | `python attack_quiz.py` |

## 사전 준비

1. **Python 환경 확인** — `python --version` (3.8 이상)
2. **패키지 설치** — `pip install pandas numpy scikit-learn plotly streamlit`
3. *(선택)* **Ollama 서버 실행** — 3교시 미리보기에서 사용 (없어도 괜찮음)

## 실습 순서

### 실습 1: attack_examples.py
- 5가지 공격 유형(SQL Injection, XSS, Command Injection, Path Traversal, Brute Force)의 실제 HTTP 요청 예시를 확인
- 간단한 규칙 기반(키워드 매칭) 탐지기를 직접 만들어보고 한계를 체감
- **관찰 포인트**: 규칙 기반은 패턴이 조금만 변형되면 탐지 실패 → 데이터 분석/ML이 필요한 이유

### 실습 2: attack_quiz.py
- 10개의 HTTP 요청을 보고 공격 유형을 맞히는 퀴즈
- 1교시 이론(웹 공격 유형)을 복습하는 시간
- 대화형으로 진행되며, 정답과 해설이 즉시 제공됨

## 1교시 핵심 정리

| 공격 유형 | 핵심 키워드 |
|-----------|-------------|
| SQL Injection | `'`, `OR`, `UNION SELECT`, `--`, `DROP TABLE` |
| XSS | `<script>`, `alert(`, `onerror=`, `<iframe>` |
| Command Injection | `;`, `|`, `&&`, `cat /etc`, `rm -rf` |
| Path Traversal | `../`, `%2f`, `/etc/passwd` |
| DDoS / Brute Force | 대량 요청, 짧은 간격, 반복 패턴 |
